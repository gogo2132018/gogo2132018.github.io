linux

linux

#1.操作系统
计算机 = 硬件 + 软件
硬件五大部件：运算器、控制器(两个统称cpu)、存储器、输入设备、输出设备(两个统称IO)

软件 = 系统软件 + 应用软件

操作系统是最为重要的系统软件，人和计算机硬件交互的界面和接口，字符界面和图形界面

Xerox - 施乐

Macintosh — macOS — iOS — watchOS

Unix —

Shell程序 - 壳程序 - BASH(Bourne Again SHell)

man - manual - 查看帮助手册
URL - Universal Resource Locator - 统一资源定位符
URI - Universal Resource Identifier - 统一资源标识符

linux系统有7个初始化级别
0 - 关机
1 - 单用户模式
2 - 多用户模式
3 - 多用户网络模式
4 - 保留
5 - 图形化
6 - 重启

SSH/TCP - 22号端口 - FinalShell(windows)/Termius(macOX)

公网IP地址、用户名(root)、密码

自动补全 - tab

w / who / whoami / last
whatis / man / –help / whereis
clear / history / !编号 / !! logout / exit
shutdown / reboot /

/ - 根目录 - 相当于Windows下计算机
/home/helloworld - 普通用户的主目录再/home
pwd - 查看当前工作目录 - print working directory
ls - 查看文件和文件夹 - list directory contents
-a 查看隐藏文件夹 - all
-l - 长格式查看 - long
mkdir - 创建文件夹 - make direcroty
-p - 在需要的时候创建父文件夹 - parents
rmdir - 删除空文件夹 - remove empty directory
cd - 切换目录 - change directory
- 绝对路径 - 以根路径/作为访问的基路径
- 相对路径 - 以当前路径作为访问的基路径
- ~ - 当前用户的用户主目录，如果是root用户就是/root
touch - 如果文件不存在就创建空文件，如果文件存在就修改文件的最后访问时间
rm - 删除文件 - remove
-r - 递归式的删除(用于删除文件夹) - recursive
-f - 强制删除(不提示) - force
-i - 交互式删除(询问是否删除，默认) - interactive
cp - 拷贝文件 - copy
-R - 递归式的复制(用于复制文件夹) - recursive
mv - 移动文件 - move
cat - 链接多个文件的内容并显示 - concatenate(倒着看tac)
head - 查看文件的开头部分
tail - 查看文件的结尾部分
more - 一屏一屏的看(less相同)
od - 二进制查看
rev - 反转一行
grep - 搜索文件的内容
-E - 使用正则表达式进行匹配
find - 搜索文件
-name - 根据文件名搜索
-size - 根据文件大小搜索
-atime - 根据文件最后访问时间搜索
-ctime - 根据文件创建时间搜索
mtime - 根基文件修改时间搜索
-exec - 执行额外的操作
“find . -name “*.html” -exec rm -f ‘{}’ ;“

阿里云服务器 —-> Web服务器(支持浏览器请求)
Apache / Nginx

安装软件
1.包管理工具 - yum - 需要链接到仓库，只能安装仓库中给的东西
yum search - 搜索仓库中的软件包
yum -y(询问全部回答yes) install …… - 安装软件包
yum remove/erase - 删除软件包
yum upgrade - 更新软件包
yum update - 更新所有的软件包
yum list installed - 查看已经安装的软件包
yum info - 查看软件包的信息
2.源代码构建安装
~ wget 下载源代码
~ 解压缩和解归档
gunzip / xz -d
tar -xvf / tar -cvf
~ 添加必要的依赖项 - yum（非必须）
~ 构建前的配置 - configure —> Makefile
- male clean
- make
- make install
~ 构建和安装 - make && make install
~ 配置必要的环境变量 - export / 修改配置文件 （非必须）

启动/停止服务
systemctl start nginx - 启动
systemctl restart nginx - 重启
systemctl stop nginx - 停止
sysetmctl status nginx - 查看状态
systemctl enable nginx - 开机自启
systemctl disable nginx - 禁用开机自启

vim - 文本编辑神器 - visual modified
命令模式 - 按键都是对应的命令
编辑模式 - 可以编辑内容
末行模式 - 可以输入命令配置vim

python的包管理工具 - pip

版本控制工具 - git

自己研究记录

mac从将本地文件上传至服务器

scp 本地文件路径 root@服务器公网地址:服务器存储路径

scp /Users/yanhan/Documents/python/python_study/Python_Cookbook/file_data/homewrok2.py root@39.107.227.235:/root/pythonCode
